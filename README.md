Multi-Agent Robot Delivery Optimization using IBM watsonx

This project demonstrates an AI-powered system for optimizing delivery robot assignments using a multi-agent architecture.

## Features
- Knowledge Synthesizer
- Decision Gatekeeper
- Dynamic Re-planner
- IBM watsonx AI explanations

## Technology
- Python
- Streamlit
- IBM watsonx.ai
- IBM Granite foundation model

## Demo
The system selects the optimal delivery robot based on real-time factors such as battery level, distance, and traffic conditions. It dynamically adapts to urgent scenarios and re-optimizes decisions. Using IBM watsonx (Granite foundation model), the system generates a clear business-friendly explanation of why a specific robot was selected.
